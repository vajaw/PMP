# TT
```
独立估算
  独立于项目团队之外的专业人士或机构对项目成本、进度、资源进行估算。这些独立估算者有专业知识和丰富的经验，能客观公正的对项目进行评估。
  收到供应商的建议书后，可以使用独立估算确保最低报价在市场范围内。让专业的人吧市场上报价算清楚。
```

# 合同
```
总价类合同 （适合不出现重大范围变更的情况下使用）
  固定总价类合同（FFP） 价格确定后不再变
  总价加激励费合同（FPIP）
    允许绩效偏离，如果目标实现，给予奖励。
    会设置价格上限，高于价格上限的全部成本由卖方承担。
  总价加经济价格调整合同（FPEPA）
    卖方履约时间跨越几年，或以不同货币支付价款。比如应对通货膨胀。
    适用于持续时间长的项目，允许根据条件变化调整合同。
成本补偿类合同（适合容易出现重大范围变更的情况下使用）
  成本加固定费用合同（CPAF）
    为卖方报销成本（合同工作中一切可列支成本）
    项目初始阶段，会支付一笔固定费用，百分比。
  成本加激励费用合同（CPIF）
    为卖方报销成本（合同工作中一切可列支成本）
    项目达到绩效目标，支付激励费用
  成本加奖励费用合同（CPFF）
    为卖方报销成本（合同工作中一切可列支成本）
    甲方主管认为绩效目标达到要求，才会支付奖励费用
混合类合同
  工料合同
    无法快速编制出准确的工作说明书的情况下，扩充人员、聘请专家、寻求外部支持使用工料合同。
```

# 知识点
```
12.1 规划采购管理
  确定要自制项目还是采购
  如果采购的话采购的策略是什么采购的计划是什么
  采购项目的工作范围是什么
  供方选择的标准是什么
  是否需要聘请专家进行独立的成本估算
  最后输出招标文件
12.2 实施采购
  通过卖方的建议书结合采购文档，以投标人会议、广告、专家判断的形式选择卖方
12.3 控制采购
  监督卖方工作绩效，控制采购关系，实施必要的变更，最终关闭合同
  索赔管理
    有争议的变更叫做索赔管理，比如卖房解释台风破坏了工厂，无法交付，则需要先启动索赔调查，确认是否属实。
  绩效审查
    对质量、资源、进度和成本绩效进行测量、比较和分析，以审查合同工作的绩效。
    其中包括确定工作包提前或落后于进度计划、超出或低于预算，以及是否存在资源或质量问题。
    采购绩效审查是对供应商合同绩效进行管理的重要工具。
  
```

# 易错题
```
  需求不明确时，如果使用敏捷框架外包项目，可以使用追求 共享风险-回报合同 关系与供应商谈判协议。
  如果供应商的可交付成果质量有问题，PM应该审查采购协议，而不是质量审计。质量审计是自己项目中只质量问题进行的。
```
