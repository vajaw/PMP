# 项目管理办公室 PMO
```
PMO 是对于项目治理相关的过程进行标准化。并且促成资源、方法论、工具和技术共享的一种组织结构。
PMO范围可大可小，从为项目管理支持服务，到直接管理一个或多个项目。
PMO可能有权在每个项目的生命周期中充当重要的相关方，PMO可以：
  提出建议
  终止项目
  领导知识传递
  根据需要采取其他行动
PMO的一个主要职能是向项目项目经理提供支持：
  指导、辅助、监督、培训
  识别和制定项目管理方法，最佳实践和标准
  对PMO的全部共享资源进行管理
  制定和管理项目政策、程序、模版、其他文件（组织过程资产）
  跨项目沟通协调
  通过项目审计，监督项目管理标准、政策、程序、模版的遵守程度


PMO有多种类型
  支持型 （对项目控制程度最低）
    相当于一个项目资源库，为项目提供模版，最佳实践、培训、信息通道、经验教训。
  控制型 （对项目控制程度中等）
    不仅提供支持，还要求项目及服从。可能要求项目：
      1、采用项目管理的框架和方法论。
      2、采用特定的模版、格式或工具。
      3、遵从治理框架。
  指令型
      直接控制和管理项目。项目经理由PMO指定并向其报告。


```
